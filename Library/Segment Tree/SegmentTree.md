# SegmentTree.cppの説明

## Segment Treeについて
一点更新/区間和や一点加算/区間最大値などを求められるアルゴリズムです。より厳密にはモノイドの演算および集合の要素はセグ木に乗り、それに対応できるようにライブラリを抽象化しています。

## e関数/op関数
e関数はモノイドの単位元を返すだけの関数です。同様にop関数は二項演算を行うだけの関数です。これを外部に置くことで、問題ごとにこの関数の中身を変えるだけでセグ木の動作を変更できます。返す値や受け取る値の型も都度変えなければなりません。時にはstructで定義された構造体を演算する必要もあります。

## struct SegmentTree
中身は自分でもよくわからない部分があるのであまり解説しませんが、clear関数はdata配列を設定する関数、update関数はdata配列に対して操作を行う関数、answer関数は区間[l,r)における演算結果を返す関数です。clear関数とanswer関数は基本触りませんが、update関数に関しては加算なのか更新なのかによって書き換える必要があります。
